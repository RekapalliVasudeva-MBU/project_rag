Aether — AI Agent + RAG Desktop App

A standalone AI agent you install on Windows. It has Normal mode (general agent with tools, skills, memory, MCP) and RAG mode (grounded on your own PDFs). Paste your own OpenRouter key in the app — it never ships with mine.

  • Left sidebar: chat sessions, RAG PDFs (add/remove/rebuild), provider settings, capability toggles (skills/tools/MCP/memory), gateway control
  • Installs to %LOCALAPPDATA%/Aether with desktop + start-menu shortcuts
  • Your data (config, skills, sessions, RAG DB) lives under %APPDATA%/aether
